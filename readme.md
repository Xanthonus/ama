# Ask me anything!

Anything means *anything*. Personal questions. Money. Work. Life. Security. Code. F1. Coffee. Whatever.

To ask a question create a new issue on this repository and I'll answer. 

### Guidelines

- Ensure your question hasn't already been answered.
- Use a succinct title and description.
- Bugs & feature requests should be opened on the relevant issue tracker.
- Support questions are better asked on Stack Overflow.
- Be civil and polite.
